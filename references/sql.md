# sql

`sql` 是只读 SQL 查询入口，用于查询工作空间索引数据库。

## 数据模型

编写、审查或修正任何 SQL 前，先读取 [`data-models/sql-schema.md`](data-models/sql-schema.md)，确认表、字段、类型取值和常见关系。

## 命令详述

- 语法：`siyuan sql <statement> [--limit <n>] -w <workspace>`
- 关键参数：
  - `<statement>`：必填，SQL 语句；只取第一个位置参数，包含空格时需按当前 shell 规则加引号
  - `--limit, -l <n>`：可选，最大返回行数；默认 `100`，小于 `1` 时按 `100` 处理，SQL 自带 `LIMIT` 时优先生效
- 输出/副作用：只读；默认按列名排序后输出表格和行数，空结果输出 `No results.`，`-f json` 时返回缩进后的行对象数组
- 备注：

## 注意事项

- 该命令无子命令，且为只读查询命令。`--dry-run` 对该命令无预览差异。
- 只允许单条 `SELECT` 或 `WITH` 查询，非查询只读语句、写语句或多语句会被拒绝。
- 末尾分号后只有空白或 SQL 注释时仍按单条语句处理；分号后还有可执行语句时会按多语句拒绝。
