# sql

`sql` 是只读 SQL 查询入口，用于查询工作空间索引数据库。

## 子命令一览

- 直接调用：`sql <statement>`

## 子命令详述

| 子命令 | 用途 | 语法 | 关键参数 | 输出/副作用 | 备注 |
|---|---|---|---|---|---|
| `sql <statement>` | 执行单条只读 SQL 查询 | `siyuan sql <statement> -w <workspace>` | `<statement>`、`--limit` | 只读；默认按列名排序后输出表格和行数，`-f json` 时返回行对象数组 | 无子命令；只允许单条只读语句，非只读或多语句会被拒绝 |

## 备注

- `--limit` 默认 100；传入小于 1 的值时按 100 处理。
- SQL 语句是位置参数，包含空格时需要按当前 shell 规则加引号。
- 该命令读取 CLI 初始化后的索引数据库，不用于直接修改工作空间数据。
